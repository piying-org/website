# 路径查询

> 路径查询用于在 `valueChange`、`outputChange`、`hideWhen`、`disableWhen` 等 Action 中定位其他字段。

## 语法说明

通过 `field.get([])` 查询其他字段的配置，即使非表单控件也可以查询（只是返回的配置中没有控件引用）。

### 路径关键字

| 关键字         | 说明                                         |
| -------------- | -------------------------------------------- |
| `#`            | 查询根级配置                                 |
| `..`           | 查询父级                                     |
| `@xx`          | 查询别名为 `xx` 的控件                       |
| `["aa"]`       | 查询子级级别的名为 `aa` 的控件               |
| `["..", "aa"]` | 查询同级别的名为 `aa` 的控件（父级下的同级） |

> 注意：任何情况下 `["aa"]` 都表示查询当前级别的名为 `aa` 的控件，所以如果要查询同级别的控件，需要使用 `["..", "aa"]`。

### 数组语法

传入一个 `string[]`，比如 `['aa','bb']`，表示从左到右逐级查询。

---

## 使用示例

### 查询根级字段

```typescript
field.get(['#', 'field1']);
```

### 查询父级字段

```typescript
field.get(['..', 'enable']);
```

### 查询子级字段

```typescript
field.get(['..', 'field2']);
```

### 查询别名字段

```typescript
field.get(['@myInput']);
```

### 组合查询

```typescript
field.get(['..', 'field1', 'field2']);
```

---

## 特殊说明

### 数组项的别名

当创建了一个数组项时，别名会创建一个单独的 Map 保存。如果查询不到的情况下，才会查询父级。

### 字段不存在

如果查询的字段不存在，将返回 `undefined`。
